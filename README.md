# Projeto Final Integrador – Serviços de Rede I

## Resumo

Este projeto tem como objetivo simular a infraestrutura de rede de uma pequena empresa utilizando o Ubuntu Server como servidor principal e duas máquinas Ubuntu Desktop como clientes.

Os principais serviços implementados foram:

* **Samba:** compartilhamento de arquivos entre os computadores da rede, com diferentes níveis de permissão para cada usuário ou grupo.
* **OpenSSH:** acesso remoto seguro ao servidor por meio do protocolo SSH.
* **xRDP:** acesso remoto à interface gráfica do servidor.
* **OpenLDAP:** gerenciamento centralizado de usuários e autenticação.
* **Fail2Ban:** proteção contra tentativas de acesso não autorizadas.

A infraestrutura foi desenvolvida em um ambiente virtualizado utilizando uma máquina Ubuntu Server como servidor e duas máquinas Ubuntu Desktop para simular os computadores dos funcionários da empresa.

O objetivo principal do projeto é demonstrar a configuração, integração e funcionamento de serviços de rede amplamente utilizados em ambientes corporativos, aplicando conceitos de administração de sistemas Linux, gerenciamento de usuários, segurança e controle de acesso.
